# Pybbs(朋也社区)

为高度定制化而生的论坛

## 为什么使用它？

1. 异常详细的文档
2. 丰富多彩的主题
3. 随心所欲的开启/关闭各种集成的服务
4. 傻瓜式部署，让不懂开发的你也能搭建自己的论坛
5. 强大的API接口，为学习编程的你提供服务端环境
6. 更多功能，期待你去发现
